Hexo个人Blog初级配置

发表于 2016-12-23 | 更新于 2017-03-06 | 分类于 互联网服务工具 |

本篇主要接着前面的搭建过程，罗列下基于hexo/NexT搭建个人bolg，一些比较基本的设置内容。
主要涉及2个配置文件：

  1. hexo配置文件 （网站目录下的_config.yml），以下简称站点配置
  2. next配置文件 （themes/next目录下的_config.yml），以下简称主题配置

0x00 Hexo配置篇

hexo的网站配置部分，主要针对站点管理的一些配置，包括基本信息，路径，目录结构，还有诸如默认分类，post名字生成结构，分页参数，插件，主题等设置。
具体的可以参考： hexo configuration
虽然英文，但是很简单，我挑几个比较常用的记录以下。

01. 头像配置

配置方法：

在站点配置文件中的# site部分增加avatar项，填入头像图片的路径，可以是互联网路径，也可以是本地的。

02. 分类/标签配置

分类和标签配置主要分为两块，路径配置和目录建立。
分类和标签目录在站点配置文件中配置，默认如下：

0x01 NexT主题配置

在站点配置中启用NexT主题后（参考## 0x02 Hexo网站搭建），就能够通过NexT配置文件，增加更多的视觉效果，以及诸如评论，分享等第三方功能。
主题配置文件位置：


官方配置说明：NexT主题配置

01. 站点信息扩展 Site Information Settings

可针对站点基本信息的调整一些扩展内容。
主要包括：

  1. favicon的位置
  2. 网站关键字，可多个，逗号
  3. 时间，开始时间，2016-2017
  4. footer样式，包括图标和版权等

02. 菜单配置 Menu Settings

针对首页菜单，进行调整，包括启停，图标和路径配置。
主要包括：

  1. 路径配置，可通过注释来停用菜单
  2. 图标配置，使用FontAwesome icon

03. 侧边栏配置 Sidebar Settings

侧边栏主要包括两种，一种是站点概览，用来显示blog owner信息，配置项包括头像（是滴，这里也可以配置），社交信息外联（链接+图标）；另一种是文章目录，用来显示post内的目录，配置项包括启停，位置，显示/隐藏模式等。
主要包括

  1. 社交链接

  2. 头像

  3. ToC
  4. 显示，看英文吧，不翻译了

04. 文章配置 Post Settings

主要是post阅读展示的一系列配置，比如首页文章列表的展示字数，post元信息显示等，基本都是true/false，就不罗列了。

05. 第三方服务配置

最强大的部分，目前支持Mathjax，百度分析，百度分析，多说评论，多说分享，disqus评论，以及我们用不上的谷歌分析，CNZZ，FACEBOOK等等。
具体请参考 第三方服务集成
这里配置一下多说。

配置流程如下：

  1. 首先你要注册个多说的账号 多说开发者中心
  2. 登陆后，顶部菜单，选择我的主页，页面会显示一个你的个人网址，该网址最终以/user/一串数字结尾，记录下这串数字，它就是主题配置文件中的User ID。
  3. 选一下官网博客，主页面右上角会有当前用户图标和名字，下方有个设置，点一下进入管理后台。
  4. 顶部菜单，后台管理中增加一个匹配的站点，配置中的大部分都不需要修改，只需要将名称和首页网址录入正确，并记录下名称，对应主题配置中的shortname。这里主要是用于评论，有一些样式上的可选择项，可以自行看一下。
